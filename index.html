Self Driving Car

The Self-Driving Car project is an exciting demonstration of the power of machine learning and artificial intelligence in the field of autonomous vehicles. Using deep Q-learning, an AI agent is trained to navigate a virtual car from the upper left-hand corner to the lower right-hand corner of a grid.

Through reinforcement learning techniques, the agent learns to make intelligent decisions based on its current state and the available actions. It explores the grid, collecting experiences and gradually improving its navigation skills through trial and error. The project utilizes a neural network as the Q-function approximator to estimate the expected rewards for different actions in different states.

The visualization of the car's movements and evolving Q-values allows viewers to witness the learning process firsthand. By studying and experimenting with the code, this project offers a unique opportunity to delve into the fundamentals of reinforcement learning and its application in self-driving cars.

As part of a larger Machine Learning and AI Repository, this project serves as an educational resource for individuals interested in machine learning and AI. It provides a hands-on experience, allowing users to gain insights into the challenges and techniques involved in training autonomous agents to navigate complex environments.

Please click on the github button to learn more about the project.

Automatic News Email Sender

The automatic newsletter app is a Python script that sends daily news to a list of subscribers via email. It uses the News API to fetch the latest news articles based on users' interests and sends them in an email using Yagmail library. The app runs on a loop and sends the newsletter every day at 6:00 AM to all the subscribers in the email list. The email list is stored in an Excel file that is read by the script. The users receive a personalized email based on their interests and are greeted by their first name. To use this app, you need to sign up for a free News API account and obtain an API key. Also, you need to have a Gmail account and generate an app password to authenticate with Gmail.

Try the automatic news application today. Please click on github button for more information.
